"Estructura de Texto Sencillo" o "Texto Estructurado Básico", ya que es una combinación de formato Markdown con un estilo más simple y directo.

Este formato no sigue la sintaxis estricta de Markdown en su totalidad (por ejemplo, no usamos códigos complejos ni sintaxis de tablas), pero está diseñado para ser fácil de leer, transferible entre plataformas (Google Docs, Confluence, GitHub) y flexible.

Nombre del formato: "Texto Estructurado Básico".
Este formato se basa en los principios de Markdown pero sin la necesidad de seguir todas las reglas rigurosas, ofreciendo una estructura intuitiva con:

Encabezados (#, ##, ###)

Listas numeradas y con viñetas

Negritas y cursivas para resaltar texto

Enlaces a recursos y documentación
